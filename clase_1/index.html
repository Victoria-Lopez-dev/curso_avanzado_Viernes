Etiquetas de maquetación

En HTML se usan etiquetas estructurales como header, main, section y footer para organizar el contenido. Los elementos de maquetación genéricos como div y span también son útiles, pero no aportan significado semántico.

Landmarks particulares

article

article define contenido independiente, como un post, una noticia o una tarjeta de información.

aside

aside contiene información relacionada o complementaria, como notas, definiciones o enlaces secundarios.

Etiquetas semánticas

section

Agrupa contenido temático dentro de una página.

header

Encabezado de página o de una sección con títulos y navegación.

footer

Pie de página con créditos, enlaces y datos de contacto.

nav

Sección de navegación con enlaces a otras partes del sitio.
